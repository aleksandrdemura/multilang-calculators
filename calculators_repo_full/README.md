# Многоязычные калькуляторы (C++, Java, C#, Go, Web)

Репозиторий содержит простые и рабочие примеры калькуляторов:
- консольные версии
- графические (где применимо) — Swing (Java), WinForms (C#), Qt (пример)
- веб-реализации (статическая и серверная)
- инструкции по сборке и запуску
- GitHub Actions workflow для автоматической сборки
- GitHub Pages сайт готов в папке `docs/` (веб-версия)

Все инструкции и комментарии — на русском языке.
